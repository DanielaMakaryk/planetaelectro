# planetaelectro
Landing Page E-commerce de productos tecnológicos, desarrollada para el curso Argentina Programa 4.0 (2023)


Planeta Electro 🌍 💻 Desarrollo de una plataforma dinámica para la compra de productos tecnológicos en línea.

🚀 Características del Proyecto. Compras online: Sitio web que ofrece tecnología de punta, garantía,atención 24/7, y envíos, según las necesidades del usuario. Interfaz de Usuario Intuitiva: Maquetación moderna y limpia orientada a mejorar la experiencia de compra (UX/UI).

🛠️ Tecnologías utilizadas: HTML5 (Estructura semántica para e-commerce). CSS3 (Estilos modernos, Flexbox y diseño adaptivo). JavaScript puro (Lógica de programación, eventos y manejo de arreglos de datos).

Proyecto de desarrollo web realizado para el curso Argentina Programa 4.0 (Año 2023).
